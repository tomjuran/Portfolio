Story Sharing project

This was a project was created in November 2020

This project was built with Node.js, Express, Mongodb as a database, and googleAuth.

This web app allows you to login using your google account and you can post stories in your dashboard and view other peoples stories in the Public stories

Technologies:
  • - Node.js
  • - Express
  • - Mongodb
  • - HTML
  • - CSS
